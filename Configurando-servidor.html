PARA CONFIGURAR O SERVIDOR IREMOS UTILIZAR O node.js E O express.

NODEJS:

  1. Instale o Node.js no seu computador (https://nodejs.org/).
  2. Abra a pasta do projeto no terminal e execute o comando para instalar as dependências: npm install.
  3. Para iniciar o servidor, execute o comando: node app.js.

EXPRESS:

  1. Instale o Express no seu projeto: npm install express.
  2. Crie um arquivo chamado "app.js" para configurar o servidor.
  3. No final do arquivo, adicione a seguinte linha de código: module.exports = app;.
  4. Execute o comando: node app.js.
  5. Abra um navegador e acesse http://localhost:3000 para ver o site em funcionamento.
